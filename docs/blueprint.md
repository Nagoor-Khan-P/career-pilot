# **App Name**: CareerPilot

## Core Features:

- Job Application Dashboard: View a consolidated list of all tracked job applications with their current status and upcoming events at a glance.
- Add/Edit Application: Create new job application entries including company, role, submission date, and initial status, with the ability to modify these details.
- Application Details & Event Tracking: Drill down into individual application pages to see comprehensive details and a timeline of associated events (interviews, assessments, offers, rejections) with dates and notes.
- Add/Edit Event Details: Add, update, and remove specific interview events for any application, allowing for notes and status updates for each event.
- Local Data Persistence: All application and event data is saved and loaded from the browser's local storage, ensuring data persistence without requiring a backend database or user accounts.
- Interview Preparation Tool: A tool that offers AI-powered tips and preparation advice relevant to upcoming interview stages or specific company attributes entered, helping users tailor their approach.

## Style Guidelines:

- Primary color: A professional and reassuring deep blue (#2666D2), conveying focus and trustworthiness.
- Background color: A soft, almost white, desaturated blue (#ECF0F5), providing a clean and calm canvas.
- Accent color: A vibrant teal (#10A4CC) for highlighting calls to action, important statuses, and interactive elements, symbolizing growth and clarity.
- Headlines and Body text: 'Inter' (sans-serif) for its modern, highly legible, and objective aesthetic, suitable for conveying clear information without distraction.
- Use a set of clear, line-based icons for navigation, status indicators (e.g., submitted, interview, offer), and actions like 'add', 'edit', and 'delete', ensuring visual clarity and ease of recognition.
- Adopt a clean, card-based layout on the dashboard for easy overview of applications, with generous white space and responsive design to ensure optimal viewing across devices.
- Implement subtle, unobtrusive transitions and micro-animations for status changes, opening/closing application details, and form submissions, enhancing the user experience without being distracting.
